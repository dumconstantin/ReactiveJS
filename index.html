Welcome to ReactiveJS

There is a huge list of TO-DO's that will make this an awesome library in the near future.

ReactiveJS provides developers a reactive programming engine that makes it easy to syncronize very complex and inter-dependent modules in large applications.

Here's an example:


        var object1 = {
            foo: 'bar'
          }, object2 = {
            foo: 'baz'
          };

          // Augment the objects.
          sync(object1);
          sync(object2);

          // Define a linking between objects.
          object1.link().to(object2);

          // Modify the value of the parent object.
          object2.foo = 'qux';

          console.log(object1.foo); // This will log 'qux'.
